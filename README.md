ParcelaFácil
Aplicativo pessoal de controle financeiro. Permite registrar rendas, gastos recorrentes, compras parceladas, gastos esporádicos e reservas de dinheiro, com suporte a BRL e USD.

Funcionalidades
Dashboard — visão geral mensal: renda, gastos recorrentes, parcelas, gastos esporádicos e percentual comprometido
Gastos recorrentes — frequência mensal, semanal (por dias da semana) ou variável (sem data fixa, ex: gasolina); suporte a períodos ativos com ou sem data de fim
Compras parceladas — controle de parcelas com status pago/pendente
Gastos esporádicos — registro de gastos pontuais com data, categoria e moeda
Reservas — caixinhas de poupança com rendimento anual, depósitos/retiradas e aporte mensal automático no calendário
Categorias — categorias personalizadas para organizar gastos
Calendário — visualização mensal de vencimentos e aportes
Relatórios — resumo anual por mês
Tecnologias
Backend: Python 3.11+, FastAPI, SQLAlchemy 2, SQLite
Frontend: HTML/CSS/JavaScript puro (sem framework), servido pelo próprio backend
